# راه‌اندازی Scheduler (فعال‌سازی خودکار اشتراک)

## توکن دسترسی
```
TOKEN = dy_sch_ + md5('daneshyar_scheduler_secret_key_2025')
```

## روش ۱: Cron (هاست Linux)
در cPanel → Cron Jobs این رو اضافه کن (هر ۵ دقیقه):
```
*/5 * * * * curl -s "https://yourdomain.ir/api/scheduler.php?token=YOUR_TOKEN" >/dev/null 2>&1
```

## روش ۲: بدون cron (کافیه!)
سیستم در هر بار که یه کاربر صفحه‌ای باز کنه، اگه اشتراک زمان‌بندی‌شده‌ای داشته باشه،
خودکار فعالش می‌کنه. یعنی حتی بدون cron هم کار می‌کنه ✓

## جریان کامل:
1. کاربر پلن انتخاب می‌کنه → صفحه payment.php
2. مبلغ واریز می‌کنه + رسید آپلود می‌کنه + (اختیاری) تاریخ شروع انتخاب می‌کنه
3. ادمین در admin/receipts.php رسید رو تایید می‌کنه
4. اگه تاریخ شروع دلخواه داشت → وضعیت `approved_pending`
5. وقتی تاریخ رسید → سیستم خودکار اشتراک رو فعال می‌کنه ✓
6. اگه بدون زمان‌بندی بود → همون لحظه فعال می‌شه ✓
